# Card-Simple-black

Cartão de perfil desenvolvido com **HTML** e **CSS**, com foco em design moderno, organização visual e boas práticas de front-end.

---

## Sobre o Projeto

O **DevCard UI** é um componente de interface que simula um perfil de usuário, comum em aplicações como redes sociais, dashboards e plataformas profissionais.

Apresenta informações essenciais de forma clara, estruturada e visualmente agradável.

---

## Objetivo

Este projeto foi desenvolvido para:

- Praticar HTML e CSS  
- Aprimorar habilidades em layout e estilização  
- Trabalhar alinhamento, espaçamento e hierarquia visual  
- Criar um componente reutilizável  

---

## Funcionalidades

- Exibição de foto de perfil  
- Nome do usuário  
- Profissão ou descrição  
- Botões de ação  
- Ícones de redes sociais 

---

## Design

O design segue uma abordagem minimalista:

- Layout centralizado  
- Uso de cores suaves  
- Bordas arredondadas  
- Estrutura simples e direta 

---

## Tecnologias Utilizadas

- HTML5  
- CSS3
